TrustLedger is a sophisticated platform acting as a neutral intermediary between employers and freelancers, eliminating payment friction and ensuring quality through AI oversight.

Key Features:

Role-Based Dashboards: Users can choose to log in as either an Employer / Client or a Freelancer / Talent, with custom dashboard interfaces for each.
AI Project Manager & QA:
AI Output Logs: The system actively monitors project progress (e.g., scanning GitHub PRs and running automated test suites).
Automated Escrow Release: Cryptographically locked funds are automatically released once the AI verifies that predefined milestone criteria (like >95% code coverage) are met.
Employer Mode Features:
AI Client Strategy Generator: Employers can input their project needs and budget, and the AI automatically crafts a strategic roadmap and logical milestones with funding splits.
Freelancer Directory: A marketplace to browse top-rated freelancers, view their completion rates/specialties, and directly "Propose Contract" or "Chat".
Contract Proposals: Ability to draft rich contracts defining the scope, amount, deadlines, and funding method.
Freelancer Mode Features:
Contract Bidding: Freelancers can formulate and submit their own contract proposals/budgets against projects.
My Assigned Projects: A tracking area showing current projects, employer details, and progress (e.g., 2/5 Milestones completed).
Extensive Payment Integrations: Support for various escrow funding/payout methods including Crypto (USDC/ETH), Google Pay (with QR Code Scanning), UPI, Paytm, and even a manual Cash escrow option.
Project Governance: Both parties can track the "Total Escrow Vault" values. Employers have overrides to "Force Approve Milestone" manually or "Request Human Audit" if a dispute arises.
AI Preferences: Users can adjust the "AI Autonomy Level," ranging from strict manual approval to fully autonomous AI payout releases.
Built-in Task Management: A personalized "To-Do" list for tracking daily items alongside project tasks.
